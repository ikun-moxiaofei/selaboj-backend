# 在线考试系统使用流程

## 系统角色说明

系统包含三种用户角色：
- **管理员(admin)**：可以管理所有用户、题目、班级和考试
- **老师(teacher)**：可以创建题目、班级和考试
- **学生(user)**：可以参加班级、参加考试、查看成绩

---

## 一、老师操作流程

### 1. 注册老师账号

1. 访问系统首页，点击"注册"按钮
2. 填写注册信息：
   - 账号：自定义账号
   - 密码：自定义密码
   - 用户角色：选择"老师"
3. 点击"注册"完成账号创建

### 2. 创建题目

#### 2.1 创建选择题

1. 登录老师账号
2. 点击"题目管理" → "创建题目"
3. 填写题目信息：
   - **题目类型**：选择"选择题"
   - **题目名称**：例如"Java基础选择题1"
   - **题目描述**：例如"以下哪个是Java的基本数据类型？"
   - **标签**：选择"Java"、"选择题"
   - **正确答案**：填写正确选项，例如"A"
   - **选择题选项**：
     - 选项A：int
     - 选项B：String
     - 选项C：ArrayList
     - 选项D：HashMap
4. 点击"提交"保存题目

#### 2.2 创建编程题

1. 登录老师账号
2. 点击"题目管理" → "创建题目"
3. 填写题目信息：
   - **题目类型**：选择"编程题"
   - **题目名称**：例如"两数之和"
   - **题目描述**：例如"输入两个整数，输出它们的和"
   - **标签**：选择"Java"、"算法"
   - **正确答案**：填写预期输出，例如"3"（对于输入1 2）
   - **判题用例**：
     - 用例1：输入"1 2"，预期输出"3"
     - 用例2：输入"3 4"，预期输出"7"
     - 用例3：输入"10 20"，预期输出"30"
   - **判题配置**：
     - 时间限制：1000ms
     - 内存限制：102400KB
     - 堆栈限制：102400KB
4. 点击"提交"保存题目

### 3. 创建班级

1. 点击"班级管理" → "创建班级"
2. 填写班级信息：
   - **班级名称**：例如"软件工程2202班"
   - **班级描述**：例如"2022级软件工程专业2班"
3. 点击"提交"创建班级
4. 添加学生到班级：
   - 进入班级详情页
   - 点击"添加学生"
   - 输入学生账号，点击"添加"

### 4. 创建考试

1. 点击"考试管理" → "创建考试"
2. 填写考试信息：
   - **考试名称**：例如"Java基础测试"
   - **考试描述**：例如"测试Java基础知识"
   - **开始时间**：选择考试开始时间
   - **结束时间**：选择考试结束时间
   - **总分**：设置考试总分
3. 选择题目：
   - 点击"添加题目"
   - 从题目列表中选择题目
   - 为每道题设置分值
   - 确保题目总分与考试总分一致
4. 点击"提交"创建考试
5. 发布考试到班级：
   - 进入考试详情页
   - 点击"发布考试"
   - 选择要发布的班级
   - 点击"确认发布"

## 二、学生操作流程

### 1. 注册学生账号

1. 访问系统首页，点击"注册"按钮
2. 填写注册信息：
   - 账号：自定义账号
   - 密码：自定义密码
   - 用户角色：选择"学生"（默认）
3. 点击"注册"完成账号创建

### 2. 加入班级

1. 登录学生账号
2. 点击"班级管理" → "班级列表"
3. 查看已加入的班级
4. 如果没有加入班级，需要老师添加

### 3. 参加考试

#### 3.1 开始考试

1. 点击"考试管理" → "考试列表"
2. 查看可参加的考试
3. 点击考试名称进入考试详情页
4. 查看考试信息：
   - 考试名称、描述
   - 开始时间、结束时间
   - 总分
5. 点击"开始考试"按钮

#### 3.2 答题

**选择题答题**：
1. 阅读题目描述
2. 查看选项内容
3. 选择正确答案（单选）
4. 点击"提交答案"
5. 系统立即评分并显示结果：
   - 正确：显示"正确 +X分"
   - 错误：显示"错误"
6. 实时更新当前得分

**编程题答题**：
1. 阅读题目描述
2. 在代码编辑框中编写Java代码
3. 选择编程语言（默认Java）
4. 点击"提交答案"
5. 系统编译并运行代码：
   - 编译成功：运行代码并比较输出
   - 编译失败：显示编译错误信息
6. 系统评分并显示结果

#### 3.3 结束考试

1. 答完所有题目后，点击"结束考试"
2. 系统确认提示，点击"确定"
3. 考试结束，显示最终成绩

### 4. 查看成绩

1. 点击"考试管理" → "考试列表"
2. 找到已完成的考试
3. 点击考试名称查看详情
4. 查看考试结果：
   - 总分、得分
   - 答题记录
   - 每道题的答案和得分
   - 正确答案对比

1. 可以查看所有题目
2. 可以编辑、删除任何题目
3. 可以创建新题目

### 3. 班级管理

1. 可以查看所有班级
2. 可以编辑、删除任何班级
3. 可以管理班级成员

### 4. 考试管理

1. 可以查看所有考试
2. 可以编辑、删除任何考试
3. 可以查看所有学生的考试成绩

## 四、常见问题解答

### Q1: 如何修改已创建的题目？

A: 进入"题目管理" → "题目列表"，找到要修改的题目，点击"编辑"按钮进行修改。

### Q2: 学生如何知道考试时间？

A: 学生登录后，在"考试列表"中可以看到所有可参加的考试及其时间安排。

### Q3: 编程题如何评分？

A: 编程题通过代码沙箱自动评分：
1. 系统编译学生提交的代码
2. 运行代码并输入测试用例
3. 比较输出结果与预期答案
4. 所有测试用例通过则得分，否则不得分

### Q4: 考试中途可以暂停吗？

A: 不可以。考试一旦开始，必须在规定时间内完成并提交。

### Q5: 可以重新参加考试吗？

A: 不可以。每场考试只能参加一次，提交后无法重新作答。

### Q6: 如何查看学生的考试成绩？

A: 老师可以在"考试管理"中查看所有学生的考试成绩和答题详情。

## 六、使用建议

### 给老师的建议：

1. **题目设计**：
   - 选择题：确保选项明确，答案唯一
   - 编程题：提供清晰的题目描述和示例
   - 设置合理的分值

2. **考试安排**：
   - 提前创建考试，给学生准备时间
   - 设置合理的考试时长
   - 题目难度适中

3. **班级管理**：
   - 及时添加学生到班级
   - 定期查看学生成绩
   - 针对性辅导

### 给学生的建议：

1. **考前准备**：
   - 熟悉考试时间
   - 复习相关知识
   - 准备编程环境

2. **考试技巧**：
   - 先易后难
   - 合理分配时间
   - 仔细审题

3. **编程题注意事项**：
   - 注意代码格式
   - 处理输入输出
   - 考虑边界情况
