### 1. 两数之和
**题目描述**：输入两个整数a和b，输出它们的和。

**参考答案（Java）**：
```java
import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int a = scanner.nextInt();
        int b = scanner.nextInt();
        System.out.println(a + b);
    }
}
```

**测试用例**：
- 输入：1 2 → 输出：3
- 输入：5 7 → 输出：12
- 输入：100 200 → 输出：300

### 8. 求最大值
**题目描述**：输入三个整数a、b、c，输出它们中的最大值。

**参考答案（Java）**：
```java
import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int a = scanner.nextInt();
        int b = scanner.nextInt();
        int c = scanner.nextInt();
        int max = a;
        if (b > max) max = b;
        if (c > max) max = c;
        System.out.println(max);
    }
}
```

**或使用Math.max()**：

**测试用例**：

## 二、选择题答案（10道）

### 1. Java基本数据类型
**题目**：以下哪个是Java的基本数据类型？

**选项**：
- A. int
- B. String
- C. ArrayList
- D. HashMap

**正确答案**：A

**解析**：
- `int` 是Java的8种基本数据类型之一（byte, short, int, long, float, double, char, boolean）
- `String` 是引用类型，虽然使用频繁，但不是基本数据类型
- `ArrayList` 和 `HashMap` 都是集合类，属于引用类型

---

### 2. Java关键字
**题目**：以下哪个不是Java的关键字？

**选项**：
- A. class
- B. interface
- C. extends
- D. main

**正确答案**：D

**解析**：
- `class` 是关键字，用于定义类
- `interface` 是关键字，用于定义接口
- `extends` 是关键字，用于继承
- `main` 不是关键字，只是程序的入口方法名

---

### 3. Java访问修饰符
**题目**：以下哪个访问修饰符的访问范围最大？

**选项**：
- A. public
- B. protected
- C. default
- D. private

**正确答案**：A

**解析**：
访问修饰符的访问范围从大到小：
1. `public` - 所有类都可以访问
2. `protected` - 同一包内的类和所有子类可以访问
3. `default` - 同一包内的类可以访问
4. `private` - 只有本类可以访问

---

### 4. Java循环语句
**题目**：以下哪个是Java的循环语句？

**选项**：
- A. if
- B. switch
- C. for
- D. try

**正确答案**：C

**解析**：
- `if` 是条件语句，不是循环语句
- `switch` 是多分支选择语句，不是循环语句
- `for` 是循环语句，用于重复执行代码
- `try` 是异常处理语句，不是循环语句

Java的循环语句包括：`for`、`while`、`do-while`

---

### 5. Java数组
**题目**：以下哪个是正确的数组声明方式？

**选项**：
- A. int arr[5];
- B. int[] arr = new int[5];
- C. array int[5];
- D. int[5] arr;

**正确答案**：B

**解析**：
- A 错误：Java中声明数组不能指定长度
- B 正确：这是标准的Java数组声明和初始化方式
- C 错误：Java中没有 `array` 关键字
- D 错误：数组长度应该在 `new` 时指定，而不是声明时

---

### 6. Java字符串比较
**题目**：在Java中，比较两个字符串内容是否相同应该使用哪个方法？

**选项**：
- A. ==
- B. =
- C. equals()
- D. compare()

**正确答案**：C

**解析**：
- `==` 比较的是引用地址，不是内容
- `=` 是赋值运算符，不是比较运算符
- `equals()` 方法比较字符串内容是否相同（正确）
- `compare()` 不是String类的方法，应该使用 `compareTo()`

---

### 7. Java继承
**题目**：Java中一个类可以继承几个父类？

**选项**：
- A. 1个
- B. 2个
- C. 3个
- D. 不限制

**正确答案**：A

**解析**：
Java是单继承语言，一个类只能有一个直接父类。这是为了避免多继承带来的复杂性和歧义性（如菱形继承问题）。但一个类可以实现多个接口。

---

### 8. Java接口
**题目**：以下关于Java接口的说法，哪个是正确的？

**选项**：
- A. 接口中可以有构造方法
- B. 一个类可以实现多个接口
- C. 接口不能继承接口
- D. 接口中的方法必须有实现

**正确答案**：B

**解析**：
- A 错误：接口不能有构造方法
- B 正确：一个类可以实现多个接口，这是Java实现多继承特性的方式
- C 错误：接口可以继承接口（使用extends关键字）
- D 错误：Java 8之前，接口中的方法都是抽象方法，没有实现；Java 8之后可以有默认方法

---

### 9. Java异常处理
**题目**：以下哪个关键字用于捕获异常？

**选项**：
- A. catch
- B. throw
- C. throws
- D. finally

**正确答案**：A

**解析**：
- `catch` 用于捕获异常（正确）
- `throw` 用于抛出异常
- `throws` 用于声明方法可能抛出的异常
- `finally` 用于定义无论是否发生异常都会执行的代码块

---

### 10. Java集合框架
**题目**：以下哪个集合类是线程安全的？

**选项**：
- A. ArrayList
- B. HashMap
- C. LinkedList
- D. Vector

**正确答案**：D

**解析**：
- `ArrayList` 不是线程安全的
- `HashMap` 不是线程安全的
- `LinkedList` 不是线程安全的
- `Vector` 是线程安全的，它的方法都是同步的

其他线程安全的集合类还有：`Hashtable`、`Collections.synchronizedList()`、`ConcurrentHashMap` 等
